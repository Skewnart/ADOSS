Error codes :
	register : l'acces n'exite pas, l'utilisateur est déjà enregistré, la validation de l'acces est déjà en cours
	changepassword :	l'access n'existe pas, l'utilisateur n'existe pas, l'utilisateur n'a pas access, les identifants ne correspondent pas,
						le mot de passe est le même que l'ancien, les deux mots de passe ne correspondent pas

	user add username : l'utilisateur existe déjà, caractères spéciaux utilisés interdits ('"') [Demander le mot de passe ensuite, sans l'écrire physiquement ?]
	user delete username : L'utilisateur n'existe pas, [Demander confirmation avant de le faire]
	user active on/off username : L'utilisateur n'existe pas, l'utilisateur est déjà actif, l'utilisateur est déjà inactif [Faire gaffe pour les acces]

	access add accessname : L'acces existe déjà, caractères spéciaux interdits
	access delete accessname : L'acces n'existe pas [Demander confirmation]
	access grant accessname username : L'acces n'existe pas, L'utilisateur n'existe pas, L'utilisateur a déjà cet access [Bien enlever du pending]
	access revoke accessname username : L'acces n'existe pas, L'utilisateur n'existe pas, L'utilisateur n'a pas cet acces [Enlever du pending et de la liste]
	access revokeall username : L'utilisateur n'existe pas, l'utilisateur n'a aucun access [Enlever du pending et de la liste]
